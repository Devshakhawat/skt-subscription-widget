A simple plugin to collect users email to mailchimp

== Description ==
This plugin allows admin to collect users email by adding a simple widget. It requires API key of Mailchimp and the input filed will be shown on under settings->general page. 

== Installation ==
Simple installation like others plugin. just get it and activate.
